I created this Defile Calculator with my colleague Zhengguang Wang. This Calculator is a web application that is designed to solve the Defile problem in Hearthstone, a Boardgame of Blizzard. This Calculator takes the current board status and outputs the sequences of steps to reach a Defile-Clearable board. User can also click to see the board status lively after each step.

We employed dynamic programming to get the solution and are currently improving our Calculator's performance regarding user input by allowing user to upload a screen shot to automatically fill in values and minion status instead of doing it manually. Please feel free to play around with it and have fun!
